# Salesforce Customer Management System

## 📌 Overview

This project is a CRM-based Customer Management System built using Salesforce. It demonstrates core concepts of enterprise application development including custom object creation, backend logic implementation using Apex, and data handling within a CRM environment.

---

## 🚀 Features

* Created a custom **Customer object**
* Added fields: Email, Phone, Address
* Implemented backend logic using **Apex Classes**
* Retrieved and managed customer data
* Demonstrated CRM architecture and data modeling

---

## 🛠️ Tech Stack

* Salesforce (CRM Platform)
* Apex (Backend Language)
* REST API Concepts
* JSON & HTTP

---

## 📂 Project Structure

* `force-app/main/default/classes/` → Apex classes
* `force-app/main/default/objects/` → Custom objects and fields
* `force-app/main/default/layouts/` → UI layouts

---

## ▶️ How to Run

1. Install Salesforce CLI
2. Clone this repository
3. Login to Salesforce:

   ```
   sf org login web
   ```
4. Deploy project:

   ```
   sf project deploy start
   ```
5. Open Salesforce org:

   ```
   sf org open
   ```
